# Eduka Web Aplikacija

Ovaj repozitorijum sadrži Next.js aplikaciju udruženja Eduka, namenjenu predstavljanju programa kontinuirane edukacije, vesti i informacija o članstvu. Projekat koristi TypeScript i modularnu strukturu komponenti radi lakšeg održavanja i bržeg razvoja novih sadržaja.

## Tehnologije

- Next.js 14
- TypeScript
- Bootstrap 5

## Razvoj

Instalirajte zavisnosti i pokrenite razvojni server:

```bash
npm install
npm run dev
```
